# 赛博哈苏 CyberHassel

赛博哈苏是一个纯前端、零上传、零后端成本的小应用。它不是传统意义上的滤镜站，而是一个把普通手机照片强行塞进高奢影像语境的网页工具：用画册留白、经纬度、器材参数、流金地名、极细标尺线和人文诗句，把废片压成像国家地理边框、中画幅画册、哈苏平替那一挂的社交货币。

对应的用户心智很直接：

- 买不起几十万相机，但想白嫖哈苏的气质
- 不想装一堆 App，只想打开网页立刻出图
- 不想上传隐私照片，希望所有处理都在本地完成
- 想做一张高级感旅行卡片，而不是花里胡哨的修图拼贴

## 这个版本能做什么

- 本地读取图片，不上传服务器
- 自动提取照片主色，决定冷白或矿石黑底衬
- 尝试解析 EXIF，包括拍摄时间、器材、光圈、快门、ISO、焦距、GPS
- 有 GPS 时优先使用真实经纬度；没有时使用精心设计的默认“高奢器材语境”
- 根据坐标或文件名匹配离线景点诗句
- 生成带流金地名、机械标尺线、诗句与编号的 4:5 成片
- 一键导出 PNG，适合朋友圈、小红书、相册保存

## 本地打开方式

这是纯静态项目，最简单的方式有两种：

1. 直接双击 `index.html`
2. 或者用任意静态服务器打开

成功标志：

- 页面背景是极黑色
- 中间能看到 `丢入照片 刻录时空`
- 选择照片后，几乎立刻出现一张带边框和信息排版的成片

排查步骤：

1. 如果页面空白，先确认浏览器不是太老，建议用最新版 Chrome 或 Edge。
2. 如果拖图没反应，先点虚线框直接选图，确认图片格式是 JPG、PNG、WebP 之一。
3. 如果经纬度没出现，通常是原图本身没有 GPS 信息，这不是程序坏了，页面会自动用默认参数补位。
4. 如果导出失败，先换一张分辨率稍小的图试试，极大图片有时会吃掉浏览器内存。
